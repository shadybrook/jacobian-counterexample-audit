# Contributing

Corrections and independent checks are welcome, especially from specialists in
polynomial automorphisms, affine algebraic geometry, Weyl algebras, and Poisson
geometry.

For a mathematical correction, please include:

1. the exact proposition, equation, or line affected;
2. a countercalculation or a precise reference;
3. whether the issue changes the core determinant/collision certificate or only
   a downstream interpretation; and
4. a regression test when the issue is computational.

Run `make verify` and `make test` before proposing a code change. Priority and
attribution corrections should link to time-stamped public evidence. Do not
remove the original discovery credit or the AI-assistance disclosure.
